# Carl Janzell Oropesa — Developer Portfolio

Galaxy-themed developer portfolio showcasing my work on the UPLB HR systems ecosystem — Laravel + Filament applications integrated over Apache Kafka — and my AI-native engineering practice.

**Live site:** https://darkify19.github.io/carl-janzell-portfolio/

## Stack

- [Astro 5](https://astro.build) — static site framework
- [Tailwind CSS 4](https://tailwindcss.com) — styling
- Vanilla canvas starfield with parallax, twinkle, and shooting stars
- Deployed to GitHub Pages via GitHub Actions

## Development

```sh
npm install
npm run dev      # local dev server
npm run build    # production build to dist/
```
